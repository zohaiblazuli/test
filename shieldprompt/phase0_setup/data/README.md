# Phase 0 Data Directory

This directory contains configuration and setup data for Phase 0.

## Files

### `model_config.json`

Generated by `setup_ollama.py`. Contains:

```json
{
  "models": [
    {
      "name": "llama3.2",
      "tag": "3b",
      "full_name": "llama3.2:3b",
      "size_gb": 2.0,
      "description": "Meta Llama 3.2 3B - General baseline model",
      "vulnerability_profile": "Medium",
      "required": true
    },
    ...
  ],
  "setup_date": "2025-11-04 20:15:00",
  "total_models": 5,
  "required_models": 4
}
```

This file is used by other scripts to know which models are installed.

## Usage

This directory is automatically populated when you run:

```bash
python shieldprompt/phase0_setup/scripts/setup_ollama.py
```

Do not manually edit these files unless you know what you're doing.
